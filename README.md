# 課題名
MANY VOICES, ONE VOICE — Let’s Share What’s on Our Minds, Freely.
一人ひとり違って当然。自由にいろんな声を聴こう、皆でひとつを進めよう。 

## 課題内容（どんな作品か）

(背景)
・懇親会幹事をする度に、皆の意見を聞いてから進める幹事が増えていて助けになれないか
・部下に一つのことを聞く度に、皆の感じ方や考え方を聞いてから進めたいMgrが増えていて助けになれないか

(内容)
・多様な意見を聞きたい人が、問いを入力し、URLを渡す
・そのURLに意見を書きたい人が書くと、一目で見られるようになる
・最後にまとめ役の人は結論を書き込むことができ、皆に納得感ある意見を送ることができる
